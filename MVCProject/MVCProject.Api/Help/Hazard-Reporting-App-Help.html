How to Setup your Group?

  • Once you’ve downloaded the app, it designates you as the Admin for a new group.
  • This group can comprise 4 other members and you, allowing 5 members in total.
  • Post group creation by Admin, a unique group code is generated by the app.
  • This unique code can be shared via Copy button, which copies it to phone clipboard for sharing via text messaging or other ways (All the communication platforms).
  • Once the 4 users have used this unique group code for individual registration process to group, the group is ready to report.
  • Welcome to your own Hazard Reporting Group.
